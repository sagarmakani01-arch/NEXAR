# NEXAR

A powerful, self-hosted AI-powered development platform. Build, preview, and deploy full-stack applications with zero watermarks and unlimited usage.

## ✨ Features

- 🤖 **AI-Powered Development** - Nemotron 3 Ultra for code generation, debugging, and architecture
- 🚀 **Instant Deployment** - One-click deploy with custom subdomains, no watermarks
- 💻 **Full IDE Experience** - Monaco editor, terminal, file explorer, live preview
- 🔄 **Real-time Collaboration** - WebSocket-based multi-user editing with cursors
- 📦 **Multiple Frameworks** - React, Vue, Svelte, Next.js, Express, Full-stack templates
- 🐳 **Docker Ready** - Production-ready containerized deployment
- 🔒 **Self-Hosted** - Complete data ownership, no vendor lock-in
- ♾️ **Unlimited Usage** - No credit limits, no rate limits on AI requests
